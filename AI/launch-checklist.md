## מה שדורש אותך לפני עלייה לאוויר ⬜

### 1. דומיין ⬜
המחרוזת `DOMAIN_PLACEHOLDER` מופיעה בכל קובצי האתר (canonical, hreflang, sitemap, OG). לאחר שתבחר דומיין (למשל `eliram.net`), צריך החלפה גורפת אחת:

```bash
cd "<תיקיית הפרויקט>"
grep -rl "DOMAIN_PLACEHOLDER" . --include=*.html --include=*.xml --include=*.txt \
  | xargs sed -i '' 's/DOMAIN_PLACEHOLDER/your-domain.co.il/g'   # macOS
```
(אני יכול לבצע את ההחלפה ברגע שתגיד לי מה הדומיין.)

### 2. אחסון והעלאה לאוויר ⬜
זה אתר סטטי – האחסון הכי פשוט וחינמי:
- **Cloudflare Pages** או **Netlify** – גרירה של התיקייה או חיבור ל‑git, תעודת SSL אוטומטית, חיבור דומיין בלחיצות.
- שתיהן מתאימות מצוין; אין צורך בשרת או בבסיס נתונים.

### 3. חיבור הטופס לתיבת המייל ⬜
1. פתח טופס חינמי ב‑[formspree.io](https://formspree.io) עם הכתובת `law@eliram.net`.
2. הדבק את כתובת ה‑endpoint בקובץ `js/main.js`, בשורה:
   ```js
   const FORM_ENDPOINT = "";   // ← הדבק כאן, למשל "https://formspree.io/f/abcwxyz"
   ```
   מאותו רגע הפניות מגיעות ישירות לתיבה, והמסך היפה של "נשלח בהצלחה" נשמר.

### 4. אנליטיקס ⬜
להוסיף Google Analytics 4 (או Plausible לפרטיות) כדי לדעת מי נכנס ומאיפה. אפשר שאוסיף את ה‑snippet בכל העמודים.

### 5. Google Search Console + Business Profile ⬜
- לאמת את האתר ב‑Search Console ולשלוח את `sitemap.xml`.
- לפתוח/לעדכן כרטיס Google Business Profile למשרד (חשוב מאוד לחיפוש מקומי).
